# Catalyst

一个为 Minecraft 1.20.1 开发的客户端辅助功能模组，同时支持 Forge 和 Fabric 平台。

## 功能特性

### 自动化功能
- **自动疾跑 (Auto Sprint)** - 自动保持疾跑状态，无需双击前进键
- **自动游泳 (Auto Swim)** - 在水中自动保持游泳姿态
- **自动工具 (Auto Tool)** - 挖掘时自动切换到背包中最适合的工具，支持锁定槽位和挖掘后自动回归原槽位
- **自动武器 (Auto Weapon)** - 攻击时自动切换到背包中伤害最高的武器，支持锁定槽位
- **自动门 (Auto Door)** - 接近门时自动打开，远离后自动关闭
- **自动水桶 (Auto Water Bucket)** - 在特定情况下自动使用水桶

### 视觉增强
- **伽马值覆写 (Gamma Override)** - 调整游戏亮度，支持夜视模式，让你在黑暗中也能看清
- **透视 (Chams)** - 实体透视功能，可分别对玩家、动物、怪物开启，轻松发现隐藏生物

### 背包管理
- **背包整理 (Inventory Sorter)** - 智能整理背包物品
  - 支持3种排序模式：IPN规则排序、显示名称排序、物品ID排序
  - 支持打开容器自动整理
  - 支持整理快捷栏
- **鼠标手势 (Mouse Tweaks)** - 增强鼠标操作体验
  - 右键拖拽放置物品
  - 左键拖拽合并同类物品
  - Shift+左键快速移动物品
  - 滚轮快速移动物品

### 路径点系统
- **路径点管理 (Waypoints)** - 保存和显示世界中的重要位置标记
  - 添加/删除/编辑路径点
  - 支持传送功能（如服务器允许）
  - 3D世界渲染显示路径点位置和距离
  - 可自定义路径点颜色和名称

## 安装要求

- **Minecraft**: 1.20.1
- **Java**: 17 或更高版本
- **模组加载器**:
  - Forge: 47.4.16 或更高版本
  - Fabric: 0.18.4 或更高版本 + Fabric API 0.92.7+

## 安装方法

### Forge
1. 下载对应版本的模组文件
2. 将 `.jar` 文件放入 `.minecraft/mods` 文件夹
3. 使用 Forge 启动游戏

### Fabric
1. 下载对应版本的模组文件
2. 将 `.jar` 文件放入 `.minecraft/mods` 文件夹
3. 确保已安装 Fabric API
4. 使用 Fabric 启动游戏

## 使用方法

### 按键绑定
模组添加了以下默认按键（可在设置中修改）：
- **打开配置界面** - 可自定义所有功能的开关和设置
- **打开路径点界面** - 管理你的世界标记

### 配置说明
模组配置文件位于 `.minecraft/config/catalyst.json`，包含：
- 各功能的启用/禁用状态
- 详细功能设置（如伽马值、锁定槽位等）
- GUI 面板位置记忆

也可以在游戏中通过配置界面实时调整设置。

## 注意事项

1. 这是一个**客户端模组**，安装在服务器上不会生效
2. 部分功能（如透视）可能在某些服务器被视为作弊，请遵守服务器规则
3. 建议在单人游戏或允许使用此类模组的服务器上使用
